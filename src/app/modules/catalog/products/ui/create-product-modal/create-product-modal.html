La configuración de inventario define cómo se rastrea el stock de este producto. No puedes cambiarla una vez que el producto tenga movimientos registrados.

Stock mínimo
Cantidad mínima de stock que debe tener el producto. Cuando el stock llegue a este nivel, se mostrará una alerta para reabastecer el inventario.
Maneja inventario
Controla el stock disponible. Desactivar en servicios o productos digitales.
Maneja lotes
Rastrea lotes con número y fecha de vencimiento. Ideal para alimentos y medicamentos.
Maneja serial
Rastrea cada unidad por número de serie. Ideal para electrónicos y maquinaria.
Permitir stock negativo
Permite vender aunque no haya stock. El inventario quedará en negativo hasta que se registre la compra. Útil para carnicerías y negocios que reciben y venden antes de registrar la compra.
